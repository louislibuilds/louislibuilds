## 关于我

Helluu，我是 bubblechickenlab。:)

我是一位喜欢把想法变成实际产品的开发者。我专注于打造界面简洁、好用且易于维护的现代网页应用。

我热衷于学习新技术、探索 AI，并通过各种 side project 持续提升自己的能力。

不写代码的时候，我大概在打游戏、健身、做饭，或者计划下一趟旅行。
